# Quickstart

## Option 1: Telegram Bot (Fastest)

1. Open Telegram and go to [t.me/BigBrain2Bot](https://t.me/BigBrain2Bot)
2. Send **$0.25 USDC** on Base to the bot's wallet address
3. Type any ticker symbol (e.g. `BTC`, `AAPL`, `NVDA`, `GOLD`)
4. Receive instant analysis with buy/sell/hold recommendation

## Option 2: Virtuals Marketplace

1. Go to [app.virtuals.io/virtuals/96114](https://app.virtuals.io/virtuals/96114)

3. Pay with USDC on Base
4. Receive your deliverable within 5 minutes
